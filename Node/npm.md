1. Show npm configurations:
``` $npm config list```

2. Install package 
 * locally(../node_modules) ```npm install <package_name>```
 * globally({prefix}/lib/node/node_modules) ```npm install <pakcage_name> --global```
 
3. List local package
 * ```npm list```
